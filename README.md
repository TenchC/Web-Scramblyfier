
# Web Scramblyfier

## Install Guide
------------------
1. Open extensions page in chrome
2. Turn on developer mode (in top right)
3. Load unpacked Extension
4. Select Web Scramblyfier folder
5. Enable and pin extension

### Currently scrambly modes:

1. Everything

2. Just Divs

3. Just Headers (h1-5)

4. Just p tags

5. Custom Scrambling
